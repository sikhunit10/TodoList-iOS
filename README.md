# TodoListMore - Modern iOS Todo List App

<p align="center">A feature-rich task management application built with SwiftUI and CoreData for iOS/iPadOS. TodoListMore helps you organize tasks with categories, priorities, and due dates in a clean, intuitive interface.</p>

## Features

<p align="center">
• Create, view, edit, and delete tasks with titles and descriptions<br>
• Mark tasks as completed with a satisfying animation<br>
• Swipe actions for quick task completion and deletion<br>
• Batch operations for managing multiple tasks at once<br>
• Prioritize tasks with Low, Medium, and High levels<br>
• Set and track due dates with visual indicators for upcoming deadlines<br>
• Create custom color-coded categories<br>
• Filter tasks by status (all, today, upcoming, completed)<br>
• Search tasks by title and description<br>
• Local storage with CoreData<br>
• Dark and light mode support<br>
• Responsive layout for iPhone and iPad<br>
• Home Screen widgets for quick access to tasks and actions
</p>

### Widgets

<p align="center">TodoListMore includes three Home Screen widgets:</p>

<p align="center">
1. <b>Today's Tasks Widget</b>: Shows tasks due today with their titles and times<br>
2. <b>Priority Tasks Widget</b>: Shows high priority tasks that need your attention<br>
3. <b>Quick Add Task Widget</b>: A button to quickly add new tasks straight from your Home Screen
</p>

## Technical Details

<p align="center"><b>Architecture</b></p>
<p align="center">
• <b>UI Framework</b>: SwiftUI for modern declarative interface design<br>
• <b>Design Pattern</b>: MVVM (Model-View-ViewModel)<br>
• <b>Data Persistence</b>: CoreData with optimized context management<br>
• <b>State Management</b>: Combine framework for reactive data flow<br>
• <b>Background Processing</b>: Asynchronous operations for performance
</p>

<p align="center"><b>Key Components</b></p>
<p align="center">
• <b>DataController</b>: Central service for CoreData operations and state management<br>
• <b>CategoryViewModel</b>: Dedicated view model for category management<br>
• <b>TaskListView</b>: Main interface with filtering and search capabilities<br>
• <b>TaskFormView</b>: Rich form for task creation and editing<br>
• <b>CategoryListView</b>: Management interface for categories<br>
• <b>TodoWidgets</b>: WidgetKit extensions for Home Screen widgets
</p>

<p align="center"><b>Requirements</b></p>
<p align="center">
• iOS 16.0+ / iPadOS 16.0+<br>
• Xcode 14.0+<br>
• Swift 5.7+
</p>
